# Session Notes

## Projeto
Site para catalogar jogos brasileiros, com proposta parecida com o Backloggd.

## Stack
- Docker
- PostgreSQL
- Frontend: React + Vite
- Backend: Spring

## Fase Atual
Finalizando a primeira parte do projeto:
- site funcionando com requisições entre frontend e backend;
- integração completa entre front, back e banco.

## Próxima Fase
Implementar sistema de autenticação e usuários.

## Objetivo de Curto Prazo
Fechar a base do catálogo (fluxo principal e integrações), para depois evoluir para conta de usuário, login e controle de acesso.
